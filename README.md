# Task Manager

## Описание
Этот проект представляет собой простой веб-приложение для управления задачами, реализованное с использованием Go и микрофреймворка Fiber. Приложение предоставляет возможности для создания, получения, обновления и удаления задач через HTTP API.

## Установка и запуск

### Требования
- Go 1.16+
- Модульная система Go (go.mod)

## API

### Создание задачи
- **URL:** `/tasks`
- **Метод:** `POST`
- **Тело запроса:**
    ```json
    {
        "description": "string",
        "deadline": "int64"
    }
    ```
- **Ответ:**
    ```json
    {
        "id": "int64"
    }
    ```

### Получение задачи
- **URL:** `/tasks/:id`
- **Метод:** `GET`
- **Ответ:**
    ```json
    {
        "id": "int64",
        "description": "string",
        "deadline": "int64"
    }
    ```

### Обновление задачи
- **URL:** `/tasks/:id`
- **Метод:** `PATCH`
- **Тело запроса:**
    ```json
    {
        "description": "string",
        "deadline": "int64"
    }
    ```
- **Ответ:** `200 OK`

### Удаление задачи
- **URL:** `/tasks/:id`
- **Метод:** `DELETE`
- **Ответ:** `200 OK`

## Логирование
Логирование осуществляется с использованием библиотеки Logrus. Логи пишутся в стандартный вывод.
